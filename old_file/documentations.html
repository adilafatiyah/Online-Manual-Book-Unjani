Introduction

Aplikasi X-Gracias terbagi menjadi tiga modul, yaitu Modul admisi (digunakan untuk registrasi ulang mahasiswa baru secara online dan onsite), Modul akademik (digunakan untuk kelola proses akademik baik untuk mahasiswa, dosen, BAA, dan Kaprodi), dan Modul HR (digunakan untuk kelola data pegawai pada Institut Bisnis dan Informatika Kesatuan Bogor).

Modul Admisi

Berikut ini adalah aktor dan alur proses dari Modul admisi:

MABA Mahasiswa baru wajib melakukan register ulang pada Modul admisi setelah melakukan pelunasan pembayaran tagihan biaya pendidikan. Kemudian mahasiswa baru wajib melakukan register onsite di IBIK Bogor.
Admisi Admisi wajib melakukan validasi registrasi ulang baik secara online maupun onsite
Keuangan Keuangan wajib melakukan validasi pelunasan biaya pendidikan terhadap mahasiswa yang sudah melakukan pembayaran.
BAA BAA wajib melakukan generate kelas

Data Mahasiswa

Berikut ini adalah fitur dari menu data mahasiswa:

Jumlah Mahasiswa Menu ini terdiri dari beberapa menu yaitu: jumlah mahasiswa program studi per semester, jumlah mahasiswa, jumlah mahasiswa transfer ekstensi, jumlah mahasiswa program studi, jumlah detail mahasiswa program studi, distribusi agama mahasiswa, distribusi jenis kelamin mahasiswa, distribusi provinsi mahasiswa, distribusi suku mahasiswa, distribusi pekerjaan orang tua, distribusi pendapatan orang tua, rekap status mahasiswa universitas, rekap status mahasiswa fakultas, rekap status mahasiswa program studi, rekap Kelulusan tepat waktu.
Surat Keterangan Lulus Menu ini menampilkan surat keterangan lulus berdasarkan filter program studi, angkatan, periode yudisium, dan mahasiswa.
Presentase Kelulusan Menu ini menampilkan daftar persentase kelulusan, data yang ditampilkan antara lain: angkatan, persentase kelulusan tidak tepat waktu, dan persentase kelulusan tepat waktu.

Pembayaran Mahasiswa Menu ini menampilkan daftar pembayaran mahasiswa,.

Aktor Menu Cuti

Berikut ini adalah aktor proses dari menu menu cuti:

Admin SDM Admin SDM bertugas untuk melakukan kelola menu cuti dan monitoring cuti pegawai.
Pegawai Pada menu ini pegawai dapat melakukan melengkapi data cuti pegawai.
Atasan Pada menu ini atasan perlu untuk approval cuti pegawai.

Kelola dan Monitoring Data Cuti Pegawai

Menu ini digunakan untuk kelola dan monitoring cuti pegawai oleh admin SDM, berikut tatacara untuk melakukan kelola dan monitoring cuti pegawai:

  • Tata cara generate kuota cuti:
  • 1. Scrolldown menu manajemen SDM
  • 2. Scrolldown menu cuti
  • 3. Akses menu pengaturan cuti
  • 4. Isi Data
  • 5. Klik button generate

Pengajuan Cuti Pegawai

Menu ini digunakan untuk pengajuan cuti pegawai oleh pegawai:

  • Tata cara pengajuan cuti:
  • 1. Scrolldown menu manajemen SDM
  • 2. Scrolldown menu cuti
  • 3. Akses menu pengajuan cuti
  • 4. Isi Data

Approval Cuti Pegawai

Menu ini digunakan untuk approval pengajuan cuti pegawai oleh atasan:

  • Tata cara approval cuti:
  • 1. Scrolldown menu manajemen SDM
  • 2. Scrolldown menu cuti
  • 3. Akses menu data cuti bawahan
  • 4. klik button approval
